### 완성된 자동화 요약

**작동 순서**

1. **Gmail – Watch emails (트리거)**: 새 메일이 오는지 15분마다 감시한다.
2. **Router (갈림길)**: 들어온 메일을 두 갈래로 나눈다.
3. **1st 갈래 (중요)** — "긴급" 키워드 필터가 걸린 길:
   - Gmail – Update email labels: 메일에 **긴급 라벨** 붙이기
   - Slack – Send a Message: 슬랙으로 **알림** 보내기
4. **2nd 갈래 (나머지 전부)** — Fallback 길:
   - Gmail – Move an email: **휴지통으로 이동**

**요구사항 충족 체크**

-  **Trigger 1개**: Watch emails
-  **Action 2개 이상**: 라벨 붙이기 + 슬랙 알림 + 휴지통 이동 (총 3개)
-  **조건 분기 1개**: Router로 중요 / 나머지 갈래 분기

**분기 경로 확인**

1. **중요 메일일 시**

### 완성된 Zapier 자동화 요약

**작동 순서**

1. **Gmail – New Email (트리거)**: 새 메일이 오는지 2분마다 감시.
2. **Paths – Split into paths (갈림길)**: 들어온 메일을 두 갈래로 보낸다. Make의 Router와 같은 역할.
3. **Path A (중요)** — "긴급" 키워드 조건이 걸린 길:
   - Gmail – Add Label to Email: 메일에 **긴급 라벨** 붙이기
   - Slack – Send Channel Message: 슬랙으로 **알림** 보내기
4. **Fallback (나머지 전부)** — 다른 갈래에 안 걸린 메일이 가는 길:
   - Gmail – Delete Email: **휴지통으로 이동**

**요구사항 충족 체크**

-  **Trigger 1개**: New Email
-  **Action 2개 이상**: 라벨 붙이기 + 슬랙 알림 + 휴지통 이동 (총 3개)
-  **조건 분기 1개**: Paths로 중요 / 나머지(Fallback) 분기

### 구현 과정 요약

| 단계 | Make | Zapier |
|:---:|:---:|:---:|
| **트리거** | Gmail – Watch emails (새 메일 감시) | Gmail – New Email (새 메일 감시) |
| **분기** | Router 모듈로 갈래 나눔 | Paths 로 갈래 나눔 |
| **중요 갈래** | 긴급 키워드 필터 → 라벨 붙이기 + 슬랙 알림 | Path A 조건 → 라벨 붙이기 + 슬랙 알림 |
| **나머지 갈래** | Fallback route → 휴지통 이동 | Fallback path → 휴지통 이동 |

### 항목별 비교

| 비교 항목 | Make | Zapier |
|:---:|:---:|:---:|
| **UI/UX** | 동그란 모듈을 선으로 잇는 그림판 방식. 흐름이 한눈에 보임 | 위→아래로 단계를 쌓는 목록 방식. 직관적이고 깔끔 |
| **설정 난이도** | 직관적. 정규식이나 라벨ID등 알아야 할 부분 있음 | 쉬움. 클릭 위주 |
| **분기(라우터) 기능** | Router가 무료 플랜에도 포함 | Paths는 Professional 플랜 이상에서만 제공 |
| **무료 플랜 범위** | 월 1,000 크레딧, 시나리오 2개, 라우터·필터 사용 가능, 15분 간격 | 월 100 tasks, 2단계 Zap만, 분기·필터 불가 |
| **연동 서비스 범위** | 3,000개 이상 앱 | 9,000개 이상 앱(업계 최다) |
| **유료 시작 가격** | Core 약 $10.59/월(연간)부터 | Professional 약 $19.99/월(연간)부터 |

### 장단점 정리

| | 장점 | 단점 |
|:---:|:---|:---|
| **Make** | 분기가 무료, 가격 저렴, 복잡한 로직에 강함, 흐름이 시각적으로 명확 | 배우기 까다로움(정규식·ID 등), 트리거·필터까지 크레딧 소모, 앱 수 상대적으로 적음 |
| **Zapier** | 만들기 쉬움, 앱 연동 최다, 트리거·필터 무료 카운트, 초보 친화적 | 핵심 분기(Paths)가 유료, 전반적으로 비쌈, 복잡한 로직엔 한계 |

### 도구 별 적합한 상황

**Make가 적합한 경우:** 자동화를 여러 개, 복잡하게 돌릴 계획이고, 비용을 아끼고 싶으며, 학습에 시간을 좀 투자할 수 있는 사람. 이번처럼 분기(Router)가 꼭 필요한 작업을 무료로 만들고 싶을 때 좋다.

**Zapier가 적합한 경우:** 자동화가 처음이거나, 빠르고 쉽게 만들고 싶고, 연동할 앱이 많은 경우.
